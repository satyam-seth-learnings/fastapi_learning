- [FastAPI Installation Doc](https://fastapi.tiangolo.com/#installation)

- Create python virtual environment

    ```bash
    python3 -m venv .venv
    ```

- Activate the virtual environment

    ```bash
    source .venv/Scripts/activate
    ```

- Install FastAPI

    ```bash
    pip install "fastapi[standard]"
    ```
