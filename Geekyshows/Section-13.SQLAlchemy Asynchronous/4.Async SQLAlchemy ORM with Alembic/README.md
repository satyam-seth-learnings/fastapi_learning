- Install SQLAlchemy with async support

    ```bash
    pip install sqlalchemy[asyncio]
    ```

- Install aiosqlite for SQLite

    ```bash
    pip install aiosqlite
    ```

- Install Alembic

    ```bash
    pip install alembic
    ```

- Initialize Alembic

    ```bash
    alembic init -t async alembic
    ```
